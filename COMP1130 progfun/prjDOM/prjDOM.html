What is the Document Object Model?


The Document Object Model, or DOM is a logical structure of documents (in this case just 2) and how those documents can be accessed and manipulated. Think of a tree shaped thinking map, except instead of putting the elements of your story in it we put the elements of your web page in it. Each level has a tag, for example the top wold be the <html> tag. The next level down would contain all of the previous levels children, for example <head>, and <body>. If you know what the DOM is and how to use it you can add, remove, and edit this map, and thus the web page itself.
